# 贡献者行为准则

## 我们的承诺

为了营造一个开放和友好的环境，我们作为贡献者和维护者承诺：
不论年龄、体型、残障、种族、性别认同、经验水平、国籍、个人外貌、宗教或性取向，
让每个人都能无骚扰地参与本项目。

## 我们的标准

**鼓励的行为：**
- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

**不可接受的行为：**
- 使用性暗示语言或图像
- 挑衅、侮辱/贬损性评论、人身攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他在专业环境中被合理认为不恰当的行为

## 执行

社区领袖负责澄清和执行我们的行为标准。如有问题，请联系项目维护者。
